How this gets live

Now — Smartsheet source of truth. Team logs each in-market program (program, client, launch date, leads, source, upcoming promos) in one Smartsheet. This page reads it and refreshes daily.
Phase 2 — Jotform feed. White-paper lead counts pull automatically from the P&I Jotform workspace (needs connector access to that team).
Phase 2 — GlobalMeet feed. Webcast registrations via scheduled export or connector once available.
Then — source of truth for lead-gen performance overall: history, trends, per-program detail.

Data plan: Smartsheet is the manual source of truth for now → native Jotform (white-paper leads) + GlobalMeet (webcast registrations) feeds in phase 2. "Upcoming promotions" is maintained by the program owner. Built off the Ad Solutions Team Dashboard pattern.
